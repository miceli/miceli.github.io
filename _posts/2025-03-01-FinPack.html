Let’s pack

R has a huge set of packages, an unprecedent open source community building packages for almost every branche of statistical computing and its applictaions. The main source of package is cran and an unenumerated number spread over github, gitlab and r-forge.

The cran task views has a very useful categorization for applications subjects. The list below is of similar task but much, much more limited and directed to map packages useful for finance, economics and econometrics applications. The preference is to list mature packages and projects with good documentation and examples. The last tag lists some data source for financial market data and database tools.
